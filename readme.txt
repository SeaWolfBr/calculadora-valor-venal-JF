# Calculadora de Valor Venal do Imóvel (PJF)

Este projeto é uma calculadora interativa que simula o valor venal de um imóvel com base nos dados fornecidos no carnê do IPTU de Juiz de Fora (MG), segundo a metodologia da Prefeitura.

---

## 📌 Funcionalidades

- Cálculo do valor venal do terreno e da edificação
- Aplicação de fator de comercialização por tipo de imóvel
- Visual responsivo com Bootstrap 5
- Máscara estilo calculadora para inserção de valores 💰
- Total compatível com dispositivos móveis
- Publicado via GitHub Pages

---

## ✅ Como usar

1. Preencha os campos conforme o carnê do IPTU
2. Clique em **Calcular Valor Venal (PJF)**
3. Veja o resultado detalhado da avaliação
